## How to use

### Install

::: code-group

@codetab pnpm

```bash
pnpm add -D vuepress-plugin-sitemap2@next
```

@codetab yarn

```bash
yarn add -D vuepress-plugin-sitemap2@next
```

@codetab npm

```bash
npm i -D vuepress-plugin-sitemap2@next
```

:::

### Usage

::: code-group

@codetab TS

```ts
// .vuepress/config.ts
import { sitemapPlugin } from "vuepress-plugin-sitemap2";

export default {
  plugins: [
    sitemapPlugin({
      // your options
    }),
  ],
};
```

@codetab JS

```js
// .vuepress/config.js
const { sitemapPlugin } = require("vuepress-plugin-sitemap2");

module.exports = {
  plugins: [
    sitemapPlugin({
      // your options
    }),
  ],
};
```

:::
